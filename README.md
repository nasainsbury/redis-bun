# redis-bun

An attempt to make a bun-redis client that interfaces directly with redis via TCP.

This is just a fun project to learn more about:

- The redis protocol
- TCP and handling connections, data streams etc
- Bun!

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To test:

```bash
bun test
```
